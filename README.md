A partir del Modelo Matemático de Análisis Predictivo, Simple Linear Regression: Implementar dicha técnica orientada a objetos.

Entradas de la Aplicación:
Un conjunto de observaciones o DataSet: (x1,y1), (x2,y2), ..., (xn,yn)
El DataSet estará hardcoded.
Un valor X  para predecir el respectivo valor Y.
NO SE CAPTURAN VALORES DE ENTRADA.

Salidas de la Aplicación:
Imprimir la Ecuación de Regresión: y = beta0 + beta1 x1,  reemplazando los valores óptimos, calculados, para los parámetros Beta.
A partir de un valor X de entrada calcular e imprimir en la terminal del computador su respectivo valor Y predecido. 
Calcular e imprimir los coeficientes de correlación y determinación.
Realizar e imprimir cinco predicciones con datos que no pertenecen al Data Set
